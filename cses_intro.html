  • Tower of Hanoi : Let us call the three sticks source, inter, and target. Approaching it using recursion, if we have a strategy to transfer n−1n−1 disks from stick AA to stick BB using an intermediate stick, then we first transfer the n−1n−1 smallest disks to inter using target. Then we push the biggest onto target and again move the top n−1n−1 to target using source as the new intermediate stick.
  • Split into 2 groups with Minimal distance: The solution does not seem obvious at all. One could try sorting and then using the reverse and add technique to make groups on both sides. But the number of elements in the groups may not be equal. Since the contraints are not strict, we can afford trying every subset possible and evaluating the minimum value. The one closest to S/2S/2 where SS is the sum will clearly win.
    min |x−(S−x)|=|2x−S|min |x−(S−x)|=|2x−S|
  • Permutations with Repititions: This problem is an interesting one. We ae to generate permutations of a string with repititions. This is related to the general Idea of avoiding repitions that is described in the Neetcode problems on Backtracking. The general idea in permuting is to make the decision of “swapping” elements, and then proceeding to generate all other permutations from a particular index onwards.
