Backtracking Ideas in Neetcode:-

  • Word Search. Usually the standard recursion problem provides you obvious sources of making the problem recursive. If one can’t make a problem recursive, it is useful to introduce a specialization of the problem. For example, searching for a word in a grid of characters does not seem recursive naturally, but instead searching for a word starting from index \(i\) in the grid starting from a position \((r,c)\), seems more doable.DFS, is an ineherently recursive algorithm. Remember to mark vertices as seen before calling the recursion, furthermore, each call of dfs will have a separate grid, so it is passed by value!. Sometimes you can mark a position on the grid with a ‘#’ to denote you have seen that one before and then pass the board by value, instead of maintaining a global visited array.
